# Bank-Loan-Risk-Profitability-Analysis (Using Excel & SQL Server)

## Project Objective
This project analyzes loan application data to evaluate lending risk and profitability using interactive Excel dashboards. It provides insights into borrower behavior, loan performance, and key financial metrics to support data-driven lending decisions.

## Dataset Used 
<a href = "https://github.com/ketan3580/Bank-Loan-Risk-Profitability-Anaysis/blob/main/Bank_loan_data_excel.xlsx">View Dashboard</a>

The dataset contains loan application records from LendingClub, including borrower details, loan amounts, credit grades, employment length, interest rates, debt-to-income ratio (DTI), and loan status.

## Tools Used 
- Microsoft Excel
- Data Cleaning & Preparation
- Pivot Tables
- Pivot Charts
- Slicers for Interactive Filtering
- KPI Cards
- Data Visualization & Dashboard Design

## Questions (KPIs) - Summary Dashboard
- What is the total number of loan applications received?
- total profit generated from issued loans?
- How does the number of loan applications correlate with the total funded loan amount?
- What is the average interest rate across all loans?
- What is the average Debt-to-Income (DTI) ratio of borrowers?

## Questions (KPIs) - Overview / Performance Dashboard
- What percentage of loans are classified as Good Loans (Fully Paid or Current)?
- What is the total funded amount for Good Loans vs Bad Loans?
- How much loan amount has been received compared to the funded amount?
- What is the distribution of loan applications by loan status (Fully Paid, Charged Off, Current)?
Do certain states show higher loan applications and better repayment rates?

## Process 
- Data Collection: Gathered and reviewed the loan dataset for analysis.
- Data Cleaning: Processed and structured the data to ensure accuracy and consistency.
- Data Analysis: Applied Pivot Tables and calculations to evaluate loan risk and profitability.
- KPI Identification: Derived key performance metrics related to loan performance and borrower behavior.
- Dashboard Development: Created interactive Excel dashboards to visualize insights and support decision-making.
